# TI Python Programs

This folder contains programs written in TI Python for TI-84 Plus CE calculators running the Python edition firmware (or with the Python app installed).

## About TI Python

TI Python is a subset of MicroPython available on compatible TI calculators. Programs are stored as `.8xv` files and edited directly on the calculator or transferred from a computer using [TI Connect CE](https://education.ti.com/en/products/computer-software/ti-connect-ce-sw).

## File Types

- **`.py`** — Python source files that can be read and edited on any computer.
- **`.8xv`** — Compiled variable files for transfer to the calculator.

## Transferring Programs to Your Calculator

1. Download or locate the `.8xv` (or `.py`) file you want.
2. Connect your calculator to your computer via USB.
3. Open TI Connect CE, then drag and drop the file onto your calculator.

The script will appear under the Python app on your calculator.
